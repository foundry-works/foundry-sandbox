## Description

Add a policy that permits sandboxes to access specified resources. If a resource matches both an allow and a deny rule, the deny rule takes precedence.

## Commands

| Command                        | Description                               |
| ------------------------------ | ----------------------------------------- |
| `sbx policy allow network`     | Allow access to network resources         |
