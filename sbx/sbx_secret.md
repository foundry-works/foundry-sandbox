# sbx secret

Manage stored secrets.

## Usage

```bash
sbx secret [COMMAND]
```

## Commands

| Command | Description |
|---------|-------------|
| `ls` | List stored secrets |
| `rm` | Remove a stored secret |
| `set` | Store a secret |

## Options

| Option | Description |
|--------|-------------|
| `-D, --debug` | Enable debug mode |
