# sbx policy ls

|          |                                              |
| -------- | -------------------------------------------- |
| Description | List active policies.                        |
| Usage    | `sbx policy ls [flags]`                      |

## Description

List active policies, showing name/ID, type, decision, and resources for each policy.

## Options

| Option       | Default | Description                                      |
| ------------ | ------- | ------------------------------------------------ |
| `--type`     | `all`   | Type of policy to show (`all` or `network`)       |
| `-D, --debug` |         | Enable debug logging                              |
