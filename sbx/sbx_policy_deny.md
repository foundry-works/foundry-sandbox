# sbx policy deny

|          |                                              |
| -------- | -------------------------------------------- |
| Description | Add a policy that blocks sandboxes from accessing specified resources. Deny rules always take precedence over allow rules. |
| Usage    | `sbx policy deny COMMAND`                    |

## Description

Add a policy that blocks sandboxes from accessing specified resources. Deny rules always take precedence over allow rules.

## Commands

| Command                        | Description                               |
| ------------------------------ | ----------------------------------------- |
| `sbx policy deny network`      | Deny access to network resources          |
