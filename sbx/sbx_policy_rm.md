# sbx policy rm

|          |                                              |
| -------- | -------------------------------------------- |
| Description | Remove a policy.                             |
| Usage    | `sbx policy rm COMMAND`                      |

## Description

Remove a specific policy rule by ID or resource.

## Commands

| Command                        | Description                               |
| ------------------------------ | ----------------------------------------- |
| `sbx policy rm network`        | Remove a network policy                   |
